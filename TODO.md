1. **Expense Split / Group Bills** - Split expenses with friends

4. **Sinking Funds** - Save for annual expenses (Christmas, school fees)
5. **Subscriptions** - Track recurring payments (Netflix, DSTV)
6. **Budget Caps** - Set spending limits per category
7. **Investments** - Track portfolio value and returns
8. **Invoices** - Generate invoices for freelancers
9. **Currency Converter** - Convert between NGN and other currencies
10. **Emergency Fund Calculator** - Calculate ideal emergency fund
11. **Wedding/Event Calculator** - Plan for special events
12. **Salary Projection** - Predict salary dates from transaction patterns
13. **Monthly Report Card** - Detailed monthly spending analysis
14. **Financial Health Score** - Comprehensive wellness score
